⚠ Firebase setup needed (3 min)

1. Go to console.firebase.google.com
Create a project → Add a Web app → copy the firebaseConfig object

2. In the left menu: Realtime Database → Create database → Start in test mode
Then go to Rules tab and set:

{ "rules": { ".read": true, ".write": true } }

3. Open this HTML file in any text editor, find const firebaseConfig = { near the top, and replace the placeholder values with your real ones. Save and re-open.

4. Open the file with ?admin=1 at the end of the URL for your admin panel. Share the plain URL with your friend.

This notice disappears automatically once a real config is detected.
